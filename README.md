# Voice-ka #
Voice-ka представляет собой десктопное приложение для голосового общения, разработанное в рамках учебного или дипломного проекта. Это мессенджер, который помимо стандартных возможностей голосового и видеосвязи предлагает пользователям уникальную функцию изменения голоса в реальном времени с помощью эквалайзеров, эффектов и микшера.

Проект рассчитан на широкую аудиторию, которая активно использует голосовые коммуникации в интернете. В первую очередь это геймеры, которым необходимо общение во время совместных игр, участники онлайн-конференций и рабочих встреч, стримеры и создатели контента, а также обычные пользователи, желающие разнообразить свое голосовое общение с друзьями и близкими. Приложение будет полезно всем, кто хочет не просто говорить, но и творчески подходить к процессу общения, добавляя индивидуальности своему голосу.

Актуальность проекта обусловлена несколькими факторами. Современные популярные мессенджеры и платформы для голосового общения, такие как Discord, Skype или Zoom, предоставляют базовые возможности для связи, но почти не предлагают инструментов для обработки и изменения голоса в реальном времени. Пользователи вынуждены прибегать к сторонним программам-виртуальным микшерам, что создает дополнительные сложности в настройке и увеличивает нагрузку на систему. Voice-ka решает эту проблему, интегрируя все необходимые инструменты обработки звука непосредственно в приложение. Кроме того, проект демонстрирует использование современного стека технологий, включая Rust и Tauri для backend-части и React с TypeScript для frontend-части, что обеспечивает высокую производительность, безопасность и надежность приложения при относительно небольшом размере исполняемого файла.

Основные цели, которые преследует проект Voice-ka, включают создание fully functional десктопного приложения с поддержкой голосовых и видеозвонков с минимальными задержками, разработку модуля обработки аудиопотока с возможностью применения различных эффектов и фильтров в реальном времени, обеспечение удобного и интуитивно понятного пользовательского интерфейса, реализацию серверной части для управления пользователями и сопряжением вызовов, а также приобретение практических навыков работы с современными инструментами разработки, включая асинхронное программирование на Rust, компонентный подход на React, интеграцию с базами данных PostgreSQL и автоматизацию тестирования с помощью Playwright и CI/CD процессов.
